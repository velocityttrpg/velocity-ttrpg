# Attributes

**Tree:** Universal

## Description

Attribute skills represent focused self-improvement — deliberate effort to strengthen a core capability rather than developing a specialized technique. Every attribute branch is available from the start with no prerequisites. The cost is opportunity: Skill Points spent improving an attribute are Skill Points not spent on specialized skills. Players who invest here are choosing raw capability over technique.

## Notes

1. Attribute skill lines have no trunk. All branches are available without a prerequisite purchase.
2. Each branch improves a single core attribute through skill purchases.
