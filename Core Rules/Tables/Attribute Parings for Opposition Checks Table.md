# Attribute Pairings for Opposition Checks Table

| Action                      | Acting Attribute   | Opposing Attribute   | Notes |
| :-------------------------- | :----------------- | :------------------- | :---- |
| Melee attack                | Strength           | Reflex               | Defender chooses Dodge, Parry, or Block. |
| Ranged attack               | Strength           | Reflex               | Defender chooses Dodge, Parry, or Block. |
| Physical / elemental spell  | Willpower          | Reflex               | ¹ Defender may only Dodge; Parry and Block cannot deflect magic. |
| Mind-affecting spell        | Willpower          | Willpower            | ² Defender resists with Willpower; no physical defense applies. |
| Grapple                     | Strength           | Strength             | |
| Shove / knockdown           | Strength           | Strength or Reflex   | |
| Escape grapple              | Strength or Reflex | Strength             | |
| Negotiation / persuasion    | Social             | Social               | |
| Intimidation / dominance    | Social             | Willpower            | |
| Charm / emotional influence | Social             | Willpower            | |
| Deception                   | Social             | Social or Willpower  | |
| Domination                  | Social             | Willpower            | |
| Glamour / illusion          | Willpower          | Willpower            | ² Illusions target the mind; resisted with Willpower. |
| Empathic influence          | Social             | Willpower            | |
| Resist poison / disease     | —                  | Fortitude            | |
| Consciousness Check         | —                  | Willpower            | |

**Footnotes**

¹ **Physical and elemental spells** (fire, lightning, force, cold, and similar): the spell occupies space — the defender can get out of the way. Dodge applies with standard rules (−2 when space is restricted, −4 when prone). Parry and Block are unavailable.

² **Mind-affecting spells** (charm, compulsion, illusion, fear, and similar): the spell targets the mind — no physical movement can deflect it. The defender resists with their dice pool + Willpower. Parry and Block are unavailable. A defender who is Stunned, Unconscious, or Incapacitated takes the hit automatically.
