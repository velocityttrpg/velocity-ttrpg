# Opposition Check Difficulty Table

Used when an Opposition Check doesn't need to be rolled — see [Opposition Checks](../Encounters/Opposition Checks.md) §"When to Make an Opposition Check". The Narrator picks a difficulty from the appropriate tier, then compares the acting character's **opposition score** (relevant Attribute + applicable flat skill bonuses, no dice) against it. Meeting or exceeding it is an automatic success; falling short is an automatic failure.

| Tier | Difficulty | Example |
| :--- | :---: | :--- |
| **Trivial** | 1–3 | An ordinary person manages this without a second thought — an unlocked door, common knowledge, a friendly chat with no real ask attached. |
| **Routine** | 4–7 | A task that rewards basic competence — a simple lock, calming a mildly annoyed shopkeeper, a short climb up a sturdy trellis. |
| **Challenging** | 8–13 | A genuine challenge requiring real training — convincing a stranger of something plausible, picking a standard lock, fording a fast creek. |
| **Dangerous** | 14–16 | Requires deep investment in the relevant area — talking down someone who has strong reason to distrust you, a quality lock, a sheer climb without gear. |
| **Extreme** | 17–20 | Near the ceiling of what a specialist can clear without rolling — swaying a hostile crowd, a vault-grade lock, scaling a crumbling cliff face in a storm. |

## Notes

1. The Narrator picks a specific value within a tier's range based on context — the low end for a more forgiving version of the challenge, the high end for a punishing one.
2. An opposition score of 0 (no relevant Attribute or applicable skills) cannot automatically succeed at anything; characters without meaningful investment must roll.
3. First draft; calibration subject to playtesting.
