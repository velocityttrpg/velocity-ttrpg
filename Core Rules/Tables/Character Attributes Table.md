# Character Attributes Table

| Attribute  | Description                                                                                                                                                          |
| :--------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Fortitude  | Physical resistance to poison, disease, exhaustion, and environmental hazards (extreme temperatures, toxic atmosphere, oxygen deprivation). Not used for Consciousness Checks. |
| Reflex     | Used when a character attempts to dodge, parry, or block. Provides a flat bonus on these defensive rolls.                              |
| Social     | Determines a character's capacity for social influence — persuasion, intimidation, negotiation, and contests of will or empathy with others. Built via the Dominance and Empathy skill-tree branches, which grant identical +Social bonuses but differ in flavor and capstone effects. |
| Strength   | Used for melee and ranged attacks and opposed physical checks (grapples, shoves, etc.). Grants 1 bonus damage die per 3 points on physical attacks.                |
| Vitality   | Determines a character's maximum Health. All characters begin with 25 Vitality. Vitality is not assigned at character creation — it increases only through skills or abilities. |
| Willpower  | Used for magical attacks and resistance to domination, intimidation, and similar Social-based effects. Grants 1 bonus damage die per 3 points on spell attacks. Used for Consciousness Checks — a dying character rolls their dice pool + Willpower. |
| Health     | Not a core attribute — a tracked value representing a character's current condition, ranging from 0 to their maximum (set by Vitality). Reduced by damage, restored by healing and recovery. |
